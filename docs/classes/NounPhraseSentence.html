NounPhraseSentence Class

Extends Sentence
Defined in: src/NounPhraseSentence.js:1
Module: entity-game

A subclass of Sentence. This class is used to represent a sentence (predicate

  • arguments) in the form of a noun. For example, "the cigarette that he was smoking".

A NounPhraseSentence can be used as an argument in another sentence.
